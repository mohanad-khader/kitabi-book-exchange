# منصة كتابي (Kitabi) - مشروع تخرج

## معلومات المشروع
**الاسم:** مهند خضر عباس أبو ندى  
**الرقم الجامعي:** 1206200821  
**المشرف الأكاديمي:** أ. أحمد محمد الأسمر  
**العام الدراسي:** 2025-2026

## نظرة عامة
منصة إلكترونية مجانية لتبادل الكتب الدراسية بين الطلاب، تهدف إلى حل مشكلة ارتفاع أسعار الكتب وتسهيل عملية التبادل بين الطلاب في فلسطين.

--- 

## المشكلة والحل
### المشكلة:
- ارتفاع أسعار الكتب الدراسية الجديدة
- صعوبة العثور على كتب مستعملة بأسعار مناسبة
- عدم وجود نظام منظم لتبادل الكتب المجانية
- صعوبة التواصل بين الطلاب الراغبين في التبادل

### الحل:
منصة مركزية تربط بين:
1. **المانحين:** طلاب يتبرعون بكتبهم مجاناً
2. **المحتاجين:** طلاب يبحثون عن كتب مجانية
3. **البائعين:** طلاب يبيعون كتبهم بأسعار رمزية

---

## الميزات الرئيسية

### للمستخدمين
- **تسجيل مجاني:** إنشاء حساب في دقيقة واحدة
- **إضافة كتب:** وضع كتب للتبادل (مجانية أو مدفوعة)
- **البحث المتقدم:** البحث بالعنوان، المؤلف، المنطقة
- **التواصل المباشر:** عبر WhatsApp مع الحفاظ على الخصوصية
- **إدارة سهلة:** متابعة الكتب المعروضة وحالتها

### للكتب
- **تصنيف حسب النوع:** مجاني أو مدفوع
- **تصنيف حسب الفئة:** جامعي، مدرسي، عام
- **تحديد المنطقة:** شمال غزة، غزة، الوسطى، خان يونس، رفح
- **تتبع الحالة:** متاح ← قيد التفاوض ← تم التبادل

## الصفحات الرئيسية

### الصفحات العامة
1. **الصفحة الرئيسية:** عرض الإحصائيات والكتب المميزة
2. **تصفح الكتب:** البحث والتصفية في جميع الكتب
3. **كيف تعمل:** شرح خطوات استخدام المنصة
4. **من نحن:** معلومات عن المنصة وأهدافها
5. **اتصل بنا:** نموذج للتواصل مع الإدارة

### الصفحات الخاصة (تتطلب تسجيل دخول)
1. **الملف الشخصي:** عرض وتعديل البيانات الشخصية
2. **إضافة كتاب:** إضافة كتاب جديد للتبادل
3. **إدارة الكتب:** تعديل وحذف الكتب المعروضة

## آلية العمل

### دورة حياة الكتاب
1. **الإضافة:** المستخدم يضيف كتاب ← حالة "متاح"
2. **الطلب:** مشتري يتواصل ← حالة "قيد التفاوض"
3. **اللقاء:** التقاء في مكان عام آمن
4. **الإتمام:** تسليم الكتاب ← حالة "تم التبادل"

### خطوات الاستخدام
1. **التسجيل:** إنشاء حساب مجاني
2. **الإضافة:** وضع الكتب للتبادل
3. **البحث:** البحث عن الكتب المطلوبة
4. **التواصل:** التواصل مع البائع عبر WhatsApp
5. **اللقاء:** التقاء في مكان عام
6. **التأكيد:** تحديث حالة الكتاب

## كيفية التشغيل

### المتطلبات الأساسية
- خادم ويب (Apache/Nginx)
- PHP 8.0 أو أعلى
- MySQL 5.7 أو أعلى
- Composer

### خطوات التثبيت
1. نسخ المشروع
2. dependencies تثبيت ال: `composer install`
3. إعداد ملف البيئة: `cp .env.example .env`
4. توليد المفتاح: `php artisan key:generate`
5. تكوين قاعدة البيانات في ملف `.env`
6. تشغيل الهجرة: `php artisan migrate`
7. تعبئة البيانات: `php artisan db:seed`
8. تشغيل الخادم: `php artisan serve`
